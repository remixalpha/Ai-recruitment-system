# Project Title

Job App

## Installation

Clone the project

```bash
  git clone https://github.com/remixalpha/Ai-recruitment-system.git
  cd Ai-recruitment-system.git
```

go to Backend

```bash
  cd backend
  rm -rf nodemodules
  npm i
  npm start
```

go to Frontend open new terminal

```bash
  cd frontend
  rm -rf nodemodules
  npm i
  npm start
```
